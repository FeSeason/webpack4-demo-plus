### webpack 4 配置的不同项目

* less作为css预处理器，可根据自己需求安装响应 loader
* 使用postcss的autoprefixer插件处理CSS浏览器兼容前缀
* 开发环境集成devServer默认端口自行修改，预置端口代理模板，按需使用
* 已配置babel，具体babel标准需求可自行配置
